# Block Dash Legal Pages

This folder is ready for free GitHub Pages hosting.

## Fast Free Setup

1. Replace `YOUR_SUPPORT_EMAIL@example.com` in:
   - `privacy-policy.html`
   - `terms-of-service.html`
   - `contact-us.html`

2. Push this repository to GitHub.

3. In GitHub, open:
   `Settings > Pages`

4. Set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`

5. Your free public URLs will look like:
   - `https://YOUR_GITHUB_USERNAME.github.io/REPOSITORY_NAME/privacy-policy.html`
   - `https://YOUR_GITHUB_USERNAME.github.io/REPOSITORY_NAME/terms-of-service.html`
   - `https://YOUR_GITHUB_USERNAME.github.io/REPOSITORY_NAME/contact-us.html`

## Store Fields

Use the Privacy Policy URL for:
- Apple App Store Connect privacy policy URL
- Google Play Console privacy policy URL

Use the Contact Us URL for:
- Apple Support URL
- Google Play developer/app support link

Terms of Service is recommended for the in-app Settings screen and support site.

## Important

This is a simple starter template, not legal advice. If you add ads, analytics, accounts, cloud saves, purchases, leaderboards, or crash reporting, update the Privacy Policy before submitting the app.
